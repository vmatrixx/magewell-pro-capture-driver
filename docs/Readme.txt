
What's here
===========
  The official Pro Capture Series Card Driver and SDK for Linux

Compatible Linux Kernel Version
===============================
  Linux kernel 2.6.35 and above
  

Compatible Device
=================
  Pro Capture AIO
  Pro Capture DVI
  Pro Capture HDMI
  Pro Capture SDI
  Pro Capture Dual SDI
  Pro Capture Dual HDMI
  Pro Capture Dual DVI
  Pro Capture Quad HDMI
  Pro Capture Quad SDI
  Pro Capture Mini HDMI
  Pro Capture HDMI 4K
  Pro Capture AIO 4K Plus
  Pro Capture HDMI 4K Plus
  Pro Capture HDMI 4K
  Pro Capture DVI 4K
  Pro Capture AIO 4k

Compatible Device Firmware Version
==================================
  Pro Capture 1.20 and above

Compiler info
=============
  GNU GCC 4.5.3 and above
